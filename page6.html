Website Development

Building a website using html code

The tool website development was used to create this website. During the tool two different approaches for building a website were offered: coding by html/CSS/javascript and working with wordpress. It was chosen to first start coding with html and to switch to wordpress if it would turn out to be too difficult using html. Switching, however, was not needed, because after quite some struggling in the beginning it did turn out to be quite doable.

As said, this site is coded by using mostly html code and a little bit of CSS and javascript. The style attributes are mostly included in the html code instead of in the CSS code, because the CSS file started to become quite big and unorganised. When implementing the style attributes in the html code it was easier to keep a clear structure.
